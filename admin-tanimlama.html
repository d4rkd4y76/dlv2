Yönetici hesabı

Aynı öğrenci girişiyle (sınıf + kullanıcı adı + şifre) uygulamaya giren kişilerde ana ekranda Yönetici butonu görünür. Veri yolu: adminPanelUsers/{sınıfId}/{öğrenciId}

Firebase kuralları: Realtime Database’de adminPanelUsers altına yazabilmek için kural eklemeniz gerekir (örnek geliştirme): 
"adminPanelUsers": {
  "$classId": {
    "$studentId": {
      ".read": true,
      ".write": true
    }
  }
}
Canlı ortamda .write’ı yalnızca güvendiğiniz kimlik doğrulamasıyla kısıtlayın.
